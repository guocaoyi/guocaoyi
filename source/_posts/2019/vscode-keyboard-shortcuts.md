# VSCode 常用快捷键

`command + p` > `shell command` 开始 `code` 命令行

<!-- more -->

```bash
# 多行转一行(折叠：改变代码结构)
Ctrl + J

# 格式化代码
Shift + Option + F # mac
Shift + Alt + F # win

# 代码折叠
Command+ Option + [ # 折叠
Command+ Option + ] # 展开

# 向上/向下 复制行
Shift + Option + Up / Down

# 拆分编辑框
Command + \ or 2,3,4
Command + 1,2,3,4 # 聚焦

# 单词选中
Command + D

# 打开关闭侧边栏
Command + B

# 跳转到特定行
Ctrl + g

# 文件中符号跳转
Shift + Option + O

# 删除上一个单词
Option + Delete

# 选中词
Option + Shift + Left / Right

# 删除行
Command + X

# 添加多个光标
Command + Option + Up / Down

# 重命名（replace）
F2

# 块选择
Shift + Option + (鼠标)

# 命令面板
# 1. 打开文件
# 2. 查看快捷键 输入 >
Command + P
```
